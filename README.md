# Store-App-Kotlin-with-Laravel-API-and-Admin-LTE-Dashboard

 * SqlLite Room DB
 * Retrofit handling http requests
 * Laravel API
 * Mysql DB 
 * Admin LTE Dashboard
 * Room
 * FCM push notifications
 * + more features
